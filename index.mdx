## Welcome

**Asignador de Zonas de Bilbao** is a static, offline-capable Progressive Web App (PWA) that automates mapping streets to their geographic zones within Bilbao.

Upload an Excel file containing street names, and the app cross-references each entry against a built-in dictionary of ~500 street-to-zone mappings. Download the enriched Excel file with a `Zona` column in seconds.
